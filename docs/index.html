系統架構

Browser ─► Next.js (single deployable)
            ├─ App Router (RSC + Client)  /  · /login · /dashboard · /projects · /settings/members
            ├─ middleware.ts ── session guard
            └─ Route Handlers (the API layer)  app/api/**/route.ts
                   │ Prisma
                   ▼
            PostgreSQL

(Full 版另外加上：API-key 驗證的公開 API /api/v1/* 與 Stripe webhook ─► 訂閱狀態)

租戶隔離在 lib/tenant.ts，授權集中在 lib/rbac.ts —— 每個 handler 都以 organizationId 過濾資料。

技術棧

Next.js 15 (App Router) TypeScript PostgreSQL Prisma ORM Auth.js (NextAuth v5) Tailwind CSS Zod

資料模型（Lite）

User · Account · Session · Organization · Membership(role) · Invitation · Project

Subscription · ApiKey · AuditLog 模型僅存在於 Full 版。
