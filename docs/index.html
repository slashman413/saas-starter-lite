Ship your SaaS, not the boilerplate.

一個達到生產標準的多租戶 SaaS 起手式：認證、組織、角色權限(RBAC)、訂閱計費、API 金鑰、稽核日誌 —— 全部內建、可直接擴充。

⚠️ 此頁為「靜態產品導覽」(GitHub Pages 無後端)。完整應用需 Vercel + PostgreSQL，clone repo 即可運行。

app.yoursaas.com/dashboard

內建功能

系統架構

Browser ─► Next.js (single deployable)
            ├─ App Router (RSC + Client)  /  · /login · /dashboard · /projects · /settings/*
            ├─ middleware.ts ── session guard
            └─ Route Handlers (the API layer)  app/api/**/route.ts
                   │ Prisma                         │ API-key auth
                   ▼                                ▼
            PostgreSQL                    Public API  /api/v1/*  ◄── external clients
                                                  Stripe webhook ─► subscription state

租戶隔離在 lib/tenant.ts，授權集中在 lib/rbac.ts —— 每個 handler 都以 organizationId 過濾資料。

技術棧

Next.js 15 (App Router) TypeScript PostgreSQL Prisma ORM Auth.js (NextAuth v5) Tailwind CSS Zod Stripe-ready

資料模型

User · Organization · Membership(role) · Invitation · Subscription · ApiKey · Project · AuditLog
